# ToDo_list
Task 4 To Do list using JS

Тема: ToDo list
Функционал:
  1. Список всех задач
      - отсортированный по новизне сверху вниз
  2. Добавлять задачи в список
      - по одной
  3. Помечать задачи как выполненые
      - любой способ
  4. Удалять задачи из списка
      - все
      - одну конкретную
